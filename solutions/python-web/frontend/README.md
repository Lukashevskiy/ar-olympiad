# Python Web Frontend

Small JS frontend for the FastAPI-backed variant.

## Pages

- `/`: scene mode with backend integration
- `/task2.html`: live Task 2 page with AR.js camera, field markers, live frame capture, backend detection, and response inspection
- `/task3.html`: live Task 3 page with AR.js camera, light marker tracking, light mode switching, and scene visualization
- `/task4.html`: live Task 4 page with AR.js camera, field markers, light marker, object capture, and responsive shadow visualization

## Run

```bash
npm install
npm run dev
```
