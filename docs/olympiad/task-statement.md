# Team AR Shadow Task

Participants build an AR system that:

1. detects four field markers and reconstructs a planar work surface;
2. detects a separate light marker and estimates light position and optional direction;
3. detects an object inside the field;
4. projects the object shadow onto the field plane;
5. supports a debug workflow for geometry validation without relying on live camera tracking.

Expected deliverables:

- marker layout reconstruction;
- object pose estimation or a documented substitute for MVP;
- dynamic shadow projection;
- explainable debug tooling and overlay outputs.

The task emphasizes engineering clarity, modularity, and the ability to iterate task-by-task.
